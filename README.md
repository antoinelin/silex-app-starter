# Installation

```sh
$ composer install
$ sudo npm install
$ find . -type d -exec sudo chmod 755 {} \;
$ find . -type f -exec sudo chmod 644 {} \;
$ bower install
$ sudo gulp

# OR one line command
$ composer install && sudo npm install && bower install && find . -type d -exec sudo chmod 755 {} \; && find . -type f -exec sudo chmod 644 {} \; && gulp
```
